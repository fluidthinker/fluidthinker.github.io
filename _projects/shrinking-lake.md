# Shrinking Lake Monitor

## Why I built this

This project explores how satellite imagery can be used to monitor changes in surface water over time. I wanted to build a workflow that was visually clear, reproducible, and useful for understanding environmental change.

## Project goal

The goal was to measure and communicate lake change using a repeatable remote sensing workflow built in Python.

## Data and tools

- Sentinel-2 imagery
- Python
- Jupyter notebooks
- GeoPandas
- Raster workflows
- Visualization outputs

## Workflow

1. Define the area of interest.
2. Gather imagery for the target dates.
3. Process the imagery to derive a water-related metric.
4. Compare conditions across time.
5. Export clear visual outputs for interpretation.
