# Trip Animation

## Why I built this

This project focused on turning time-based spatial data into a clean animation that reveals changing patterns over the course of a day.

## Project goal

The goal was to create a polished, reproducible workflow that pulled data from PostGIS, rendered map frames in Python, and combined them into a final animated output.

## Data and tools

- PostgreSQL / PostGIS
- SQL
- Python
- GeoPandas
- Matplotlib
- GIF generation workflow

## Workflow

1. Query time-based spatial data from PostGIS.
2. Build one frame at a time in Python.
3. Render consistent map images for each time slice.
4. Combine the frames into an animation.
5. Export a final visual product.
